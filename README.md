# GBCLXML
A variation of GBCL that parses a custom XML language.

## Keywords
Descriptions coming soon

~~~
cette     - <cette>
cettef    - </cette>
maisette  - <maisette>
maisettef - </maisette>
sinon     - <sinon>
sinonf    - </sinon>
sore      - <sore>
soref     - </sore>
shikashi  - <shikashi>
shikashif - </shikashi>
matawa    - <matawa>
matawaf   - </matawa>
~~~

## Strings
Any words not part of the above keywords will be taken in as strings. There is no current support for integers, pointers, and other stuff currently.
